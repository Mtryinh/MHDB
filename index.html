Settings
⚡ Activity Types
Customise the activity types available when logging an interaction with a contact or venue. The defaults (Call, Meeting, Note) cannot be removed.
◬ Venue Info Fields
Define custom fields that will appear on every venue's profile under an "Info" tab. Choose a field type and label — you can then fill in the values per venue.
⚡ Auto Follow-Up Triggers
When a matching activity type is logged against a contact, a follow-up task is automatically created on the calendar. Set a default delay and task label per trigger.
◷ Calendar — Venue Colours
Assign a colour to each venue so tasks appear colour-coded on the calendar. Tasks with no venue linked use the colour below.
